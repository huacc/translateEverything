## 遇到的问题与解决方案

### 问题1: usePDFViewer Hook测试失败
**现象**: 初始状态断言失败，期望null但返回undefined
**原因**: Hook需要options参数，测试用例未提供
**解决**: 为所有renderHook调用添加必需的options参数

### 问题2: DOMMatrix未定义错误
**现象**: PDF渲染测试报错"DOMMatrix is not defined"
**原因**: jsdom环境不支持DOMMatrix API
**解决**: 在setup.ts中添加DOMMatrix Mock实现

### 问题3: goToPage在测试中不生效
**现象**: 调用goToPage(15)后currentPage仍为1
**原因**: goToPage依赖numPages闭包，调用时numPages可能还是0
**解决**: 使用waitFor等待PDF加载完成，或使用nextPage循环调用

### 问题4: 循环调用nextPage失败
**现象**: 在单个act中循环调用nextPage，页面只前进1页
**原因**: nextPage依赖currentPage闭包，循环中状态未更新
**解决**: 将每次nextPage调用包装在独立的act中
