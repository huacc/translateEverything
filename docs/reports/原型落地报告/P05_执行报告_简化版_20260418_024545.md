## 使用说明

### 启用Mock模式
在`.env.development`中设置：
```
VITE_USE_MOCK=true
```

### 使用Mock服务
```typescript
import { taskService } from '@/services/taskService';

// 获取任务列表（自动使用Mock数据）
const tasks = await taskService.getTasks();

// 创建任务（自动使用Mock服务）
const newTask = await taskService.createTask({
  file: file,
  source_lang: 'zh',
  target_lang: 'en',
  translation_style: 'professional'
});
```
