# 后端接口协议文档

## 基础信息

- **Base URL**: `/api`
- **数据格式**: JSON
- **字符编码**: UTF-8
- **时间格式**: ISO 8601 (YYYY-MM-DDTHH:mm:ss.sssZ)

## 通用响应格式

### 成功响应
```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 错误响应
```json
{
  "code": 400,
  "message": "错误描述信息",
  "data": null
}
```

### HTTP状态码说明
- `200`: 请求成功
- `400`: 请求参数错误
- `401`: 未授权
- `403`: 禁止访问
- `404`: 资源不存在
- `500`: 服务器内部错误

## 数据类型定义

### TranslationJob (翻译任务)
```typescript
{
  id: string                    // 任务ID
  name: string                  // 任务名称
  status: JobStatus             // 任务状态
  sourceLanguage: string        // 源语言
  targetLanguage: string        // 目标语言
  progress: number              // 进度 (0-100)
  currentPage: number           // 当前页码
  totalPages: number            // 总页数
  createdAt: string             // 创建时间
  updatedAt: string             // 更新时间
  pdfUrl?: string               // PDF文件URL
  resultUrl?: string            // 翻译结果URL
}
```

### JobStatus (任务状态)
```typescript
type JobStatus = 
  | 'pending'      // 待处理
  | 'processing'   // 处理中
  | 'paused'       // 已暂停
  | 'completed'    // 已完成
  | 'failed'       // 失败
  | 'cancelled'    // 已取消
```

### TranslationSegment (翻译片段)
```typescript
{
  pageNum: number               // 页码
  blockId: string               // 块ID
  sourceText: string            // 源文本
  translatedText: string        // 翻译文本
  status: 'pending' | 'completed' | 'failed'
}
```

### QualityIssue (质量问题)
```typescript
{
  id: string                    // 问题ID
  type: 'terminology' | 'grammar' | 'style' | 'consistency'
  severity: 'error' | 'warning' | 'info'
  pageNum: number               // 页码
  blockId: string               // 块ID
  description: string           // 问题描述
  suggestion?: string           // 修改建议
}
```

| targetLanguage | string | 是 | 目标语言代码 |
| file | File | 是 | PDF文件 |

**请求示例**:
```

sourceLanguage=en
targetLanguage=zh
```

### 3. 实时事件流 (SSE)

#### 3.1 订阅任务事件
**接口**: `GET /tasks/:id/events`

**路径参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | string | 是 | 任务ID |

**请求示例**:
```
GET /api/tasks/task-001/events
Accept: text/event-stream
```

**响应格式**:
```
Content-Type: text/event-stream
Cache-Control: no-cache
Connection: keep-alive
```

**事件类型**:

##### 3.1.1 进度更新事件
```
event: progress
data: {"currentPage":5,"totalPages":20,"percentage":25}
```

**数据结构**:
```typescript
{
  currentPage: number      // 当前页码
  totalPages: number       // 总页数
  percentage: number       // 完成百分比 (0-100)
}
```

##### 3.1.2 翻译内容块事件
```
event: translation_chunk
data: {"pageNum":5,"blockId":"b3","content":"这是翻译的内容","isComplete":false}
```

**数据结构**:
```typescript
{
  pageNum: number          // 页码
  blockId: string          // 块ID
  content: string          // 翻译内容（流式）
  isComplete: boolean      // 是否完成
}
```

##### 3.1.3 执行日志事件
```
event: execution_log
data: {"stage":"translation","step":"page_5","status":"in_progress","message":"正在翻译第5页","timestamp":"2024-01-15T13:15:00.000Z"}
```

**数据结构**:
```typescript
{
  stage: string            // 执行阶段
  step: string             // 执行步骤
  status: 'pending' | 'in_progress' | 'completed' | 'failed'
  message: string          // 日志消息
  timestamp?: string       // 时间戳
}
```

##### 3.1.4 任务完成事件
```
event: task_completed
data: {"taskId":"task-001","status":"completed","message":"翻译任务已完成"}
```

**数据结构**:
```typescript
{
  taskId: string           // 任务ID
  status: 'completed' | 'failed'
  message?: string         // 完成消息
}
```

##### 3.1.5 错误事件
```
event: error
data: {"message":"翻译服务暂时不可用","code":"SERVICE_UNAVAILABLE"}
```

**数据结构**:
```typescript
{
  message: string          // 错误消息
  code?: string            // 错误代码
}
```

## 使用示例

### 完整工作流示例

#### 1. 创建任务
```javascript
const formData = new FormData();
formData.append('name', '技术文档翻译');
formData.append('sourceLanguage', 'en');
formData.append('targetLanguage', 'zh');
formData.append('file', pdfFile);

const response = await fetch('/api/tasks', {
  method: 'POST',
  body: formData
});

const { data: task } = await response.json();
console.log('任务ID:', task.id);
```

#### 2. 订阅任务事件
```javascript
const eventSource = new EventSource(`/api/tasks/${task.id}/events`);

eventSource.addEventListener('progress', (event) => {
  const data = JSON.parse(event.data);
  console.log(`进度: ${data.percentage}%`);
});

eventSource.addEventListener('translation_chunk', (event) => {
  const data = JSON.parse(event.data);
  console.log(`页${data.pageNum} 块${data.blockId}: ${data.content}`);
});

eventSource.addEventListener('task_completed', (event) => {
  const data = JSON.parse(event.data);
  console.log('任务完成:', data.message);
  eventSource.close();
});

eventSource.addEventListener('error', (event) => {
  const data = JSON.parse(event.data);
  console.error('错误:', data.message);
});
```

#### 3. 控制任务
```javascript
// 暂停任务
await fetch(`/api/tasks/${task.id}/pause`, { method: 'POST' });

// 继续任务
await fetch(`/api/tasks/${task.id}/resume`, { method: 'POST' });

// 取消任务
await fetch(`/api/tasks/${task.id}/cancel`, { method: 'POST' });
```

#### 4. 下载结果
```javascript
const response = await fetch(`/api/tasks/${task.id}/result?format=pdf`);
const blob = await response.blob();
const url = URL.createObjectURL(blob);
const a = document.createElement('a');
a.href = url;
a.download = 'translation_result.pdf';
a.click();
```

## 注意事项

1. **SSE连接管理**:
   - 客户端应处理连接断开并自动重连
   - 建议实现心跳检测机制
   - 长时间无数据时服务器会发送心跳包

2. **文件上传限制**:
   - 最大文件大小: 100MB
   - 支持格式: PDF
   - 建议使用分片上传处理大文件

3. **并发限制**:
   - 单用户最多同时处理10个任务
   - API请求频率限制: 100次/分钟

4. **超时设置**:
   - 普通API请求超时: 30秒
   - 文件上传超时: 5分钟
   - SSE连接超时: 无限制（保持连接）

5. **安全性**:
   - 所有接口需要认证（生产环境）
   - 使用HTTPS传输
   - 文件上传需要病毒扫描
   - 实施CORS策略

6. **性能优化**:
   - 任务列表支持分页和筛选
   - PDF预览使用懒加载
   - 翻译内容采用流式传输
   - 静态资源使用CDN
